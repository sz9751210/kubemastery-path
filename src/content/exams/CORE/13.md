# Jobs & CronJobs

For tasks that are intended to run to completion.

## Jobs
A **Job** creates one or more Pods and will continue to retry execution of the Pods until a specified number of them successfully terminate.

# Task 1: Create a Job
Run a one-off job that echoes "Hello Kubernetes".

```bash
kubectl create job hello --image=busybox -- echo "Hello Kubernetes"
```

```setup
# Clean up if exists
kubectl delete job hello 2>/dev/null || true
```

```verify
kubectl get job hello -o jsonpath='{.status.succeeded}' | grep 1
```

## CronJobs
A **CronJob** creates Jobs on a repeating schedule.

# Task 2: Create a CronJob
Create a cronjob that runs every minute.

```bash
kubectl create cronjob heartbeat --image=busybox --schedule="*/1 * * * *" -- echo "Beating..."
```

```setup
# Clean up if exists
kubectl delete cronjob heartbeat 2>/dev/null || true
```

```verify
kubectl get cronjob heartbeat -o jsonpath='{.spec.schedule}' | grep "*/1 * * * *"
```
