# The Pod Lifecycle

A **Pod** is the atomic unit of K8s. It wraps one or more containers.

## States
1. **Pending**: Scheduler is finding a node.
2. **ContainerCreating**: Pulling images.
3. **Running**: At least one container is up.
4. **Succeeded/Failed**: Process exited.

# Task 1: Create a Pod
Create a simple Nginx pod named `nginx-demo`.

```bash
kubectl run nginx-demo --image=nginx
```

```setup
# Clean up if exists
kubectl delete pod nginx-demo --force --grace-period=0 2>/dev/null || true
```

```verify
kubectl get pod nginx-demo --no-headers | grep Running
```
