# Application Observability

## Probes
Kubelet needs to know if your app is okay.
1.  **Liveness**: Restart if dead (deadlock).
2.  **Readiness**: Don't send traffic until ready (loading data).
3.  **Startup**: Wait for slow starts before other probes kick in.

\`\`\`yaml
livenessProbe:
  httpGet:
    path: /healthz
    port: 8080
  initialDelaySeconds: 3
  periodSeconds: 3
\`\`\`

# Task 1: Exec into Pod
List the files in `/app` (or `/`) inside a running pod named `debug-pod`.

```bash
kubectl exec -it debug-pod -- ls /
```

```setup
kubectl run debug-pod --image=nginx --restart=Never 2>/dev/null || true
```

```verify
kubectl get pod debug-pod --no-headers | grep Running
```
