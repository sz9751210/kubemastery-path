# Task 1: Multi-Container Pod
Create a pod named `ckad-wegyi3` with two containers.
Container 1: image `nginx`, name `c1`.
Container 2: image `busybox`, name `c2`, command "sleep 3600".

```setup
kubectl delete pod ckad-wegyi3 --force --grace-period=0 2>/dev/null || true
```

```verify
kubectl get pod ckad-wegyi3 -o jsonpath='{.spec.containers[*].name}' | grep c1
kubectl get pod ckad-wegyi3 -o jsonpath='{.spec.containers[*].name}' | grep c2
```
