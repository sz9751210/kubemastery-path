# Task 1: Multi-Container Pod
Create a pod named `ckad-0r6n02` with two containers.
Container 1: image `nginx`, name `c1`.
Container 2: image `busybox`, name `c2`, command "sleep 3600".

```setup
kubectl delete pod ckad-0r6n02 --force --grace-period=0 2>/dev/null || true
```

```verify
kubectl get pod ckad-0r6n02 -o jsonpath='{.spec.containers[*].name}' | grep c1
kubectl get pod ckad-0r6n02 -o jsonpath='{.spec.containers[*].name}' | grep c2
```
