# Task 1: Multi-Container Pod
Create a pod named `ckad-0l16fg` with two containers.
Container 1: image `nginx`, name `c1`.
Container 2: image `busybox`, name `c2`, command "sleep 3600".

```setup
kubectl delete pod ckad-0l16fg --force --grace-period=0 2>/dev/null || true
```

kubectl get pod ckad-0l16fg -o jsonpath='{.spec.containers[*].name}' | grep c1
kubectl get pod ckad-0l16fg -o jsonpath='{.spec.containers[*].name}' | grep c2
