# Task 1: Multi-Container Pod
Create a pod named `ckad-nprn2e` with two containers.
Container 1: image `nginx`, name `c1`.
Container 2: image `busybox`, name `c2`, command "sleep 3600".

```setup
kubectl delete pod ckad-nprn2e --force --grace-period=0 2>/dev/null || true
```
