# Scenario

You have a running pod named `nginx-pod` in the default namespace. However, it is currently isolated.

# Task

1. Verify the pod is running.
2. Expose this pod as a Service named `nginx-svc` on port 80.
3. Use `NodePort` type.

> [!TIP]
> Use `kubectl expose pod ... --type=NodePort`

```setup
# Ensure clean state
kubectl delete pod nginx-pod --force --grace-period=0 2>/dev/null || true
kubectl delete svc nginx-svc 2>/dev/null || true

# Create the pod for the scenario
kubectl run nginx-pod --image=nginx --restart=Never

# Verify scenario readiness
kubectl wait --for=condition=Ready pod/nginx-pod --timeout=60s
```

```verify
# Verify Pod
kubectl get pod nginx-pod --no-headers | grep Running

# Verify Service
kubectl get svc nginx-svc -o jsonpath='{.spec.ports[0].port}' | grep 80
kubectl get svc nginx-svc -o jsonpath='{.spec.type}' | grep NodePort
```
