# Admission Controllers

Intercepting requests to the API server before an object is persisted.

## Types
- **Mutating**: Can modify the object.
- **Validating**: Can only accept or reject.

# Task 1: Check Admission Plugins
Verify which admission plugins are enabled in the API Server.

```bash
kubectl exec -it kube-apiserver-controlplane -n kube-system -- kube-apiserver -h | grep enable-admission-plugins
```

```setup
echo "Ready"
```

```verify
kubectl get pods -n kube-system -l component=kube-apiserver > /dev/null
```
