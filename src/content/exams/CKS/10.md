# Network Policies

The firewall for Kubernetes.

> [!WARNING]
> By default, K8s is a **Flat Network**. Any pod can talk to any pod.

## Default Deny Logic
Start by denying everything, then allow specific traffic.

\`\`\`bash
# See if any policies exist
kubectl get networkpolicies
\`\`\`
