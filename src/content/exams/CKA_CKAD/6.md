# Services: Exposing Applications

## Service Types
1. **ClusterIP**: Internal only.
2. **NodePort**: Exposure on static port (30000+).
3. **LoadBalancer**: Cloud provider LB.

# Task 1: Expose Deployment
Create a deployment `my-dep-svc` and expose it as NodePort.

```bash
kubectl create deployment my-dep-svc --image=nginx
kubectl expose deployment my-dep-svc --port=80 --target-port=80 --type=NodePort --name=my-svc
```

```setup
kubectl delete deployment my-dep-svc 2>/dev/null || true
kubectl delete svc my-svc 2>/dev/null || true
```

```verify
kubectl get svc my-svc -o jsonpath='{.spec.type}' | grep NodePort
```
